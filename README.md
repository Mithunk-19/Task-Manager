# Task Manager App

A simple and interactive Task Management App built using **React**.

## Features
- 📝 Add, edit, and delete tasks.
- ✅ Mark tasks as completed.
- 🔍 Search and filter tasks.
- 🎨 Light & Dark mode.
- 📌 Drag & drop to reorder tasks.
- 📅 Stores tasks in local storage.
